# Stefan's 3D models

* A3000D LED jig - small jig to make it easier to grind down 10x5mm LEDs into
  Amiga 3000D 10x2mm LED panel size
* A3000 power button - power button plus rod for Amiga 3000D
* A3000 power button bracket - I got a case that came without this one
* A4000 Keylock socket for USB-C - Replace the KO132A-1501 PC key lock on your
  Amiga 4000D with a USB-C port for easy access to Z3660 or Kicksmash32
